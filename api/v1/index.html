Provision a company

POST /companies/{organization_number}

This tells our backend to produce a company instance, this should only be done once and if the above GET request returns 404. If this call returns 200, then you may do a GET request to fetch the company again.

Warning! This will not associate a company with your user profile, but will allow you to later fetch information about it via this API.
